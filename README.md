This is an XML/RSS reader for the GVSU Events Calander. Want to find free food around campus? This is the tool for you!

Instructions:
1. Go to the GVSU Events Calander: https://www.gvsu.edu/events/
2. Copy the current "XML/RSS feed (.xml)" which can be found at the bottom of the page
3. Paste the XML/RSS feed into the "events.xml" file
4. Optional - change the keywords to search for
5. Run!
